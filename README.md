# 在线文档项目

个人毕设
主要技术栈：Vue2.0+Egg+Mongodb

## 项目前言
团队中会遇到文档管理的需求，包括技术文档，接口文档，excel文档，和产品原型的托管等需求

## 本地开发
先`npm i`

 启动前端项目
`npm run dev-web`

启动后端服务
`npm run dev`

 打包前端项目
`npm run build-web`

## 本地部署
启动一个服务

先build个前端版本，然后再部署egg服务到本地
